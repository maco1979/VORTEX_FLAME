## ⚠️ 歌词创作警示（重要！）

### 用户反馈问题

| 问题 | 解决方案 |
|------|----------|
| ❌ "答案"重复太多 | 避免使用"答案"，寻找新的表达方式 |
| ❌ 歌词审美疲劳 | 多样化表达，不断学习新的表达方式 |
| ❌ 句式重复 | 变化句式结构，穿插长短句 |

### 歌词改进原则

1. **避免重复词**：同一首歌中同一个词不出现超过2次
2. **变化表达**：同一个意思用不同方式表达
3. **创新意象**：创造新的独特的意象，不要只用常见的词
4. **用心穿心**：歌词要打动人心，不是堆砌词汇
5. **不断进化**：每次创作都要有新的表达方式

### 禁止重复的词汇

```
答案、孤独、寂寞、悲伤、眼泪、星光、火焰（同一个词在一首歌里最多出现2次）
```

## 🎵 和声混音知识（Mixed in Key Camelot Wheel）

FKJ必须理解和声混音，为创作服务！

### 一、Camelot Wheel基础

每个调性用数字+字母表示：
- **数字1-12**：对应12个音（像时钟）
- **字母A**：小调（Minor）= 情绪化、暗淡
- **字母B**：大调（Major）= 明亮、积极

```
内圈A = 小调，外圈B = 大调
```

### 二、完整调性映射表

| Camelot | 音阶 | 情绪特点 |
|---------|------|----------|
| 8A | A Minor | 亲密、反思 |
| 8B | E Major | 明亮、自信 |
| 9A | E Minor | 紧张、电影感 |
| 9B | C# Major | 上升、期待 |
| 7A | D Minor | 忧郁、怀旧 |
| 7B | Bb Major | 快乐、庆祝 |
| 10A | A Minor | 渴望、追求 |
| 10B | F Major | 温暖、乡村 |
| 11A | D Minor | 深沉、严肃 |
| 11B | Bb Major | 结束、释放 |
| 12A | G Minor | 悲伤、失落 |
| 12B | Db Major | 庆祝、高潮 |

### 三、混音过渡规则

**1. Same Key（同调）**
- 代码完全相同：8A → 8A
- 用途：长时间同风格无缝过渡

**2. Relative Key（相对调）**
- 数字相同，A↔B切换：8A → 8B
- 情绪转折：小调暗淡 → 大调希望
- 用途：情绪高潮点

**3. Adjacent Key（相邻调）**
- 数字±1，字母不变：8A → 7A 或 9A
- 最安全的混音方式
- 用途：开场、结尾

**4. Diagonal Mix（对角线）**
- 数字±1，字母切换：8A → 9B
- 能量提升效果
- 用途：打破单调

### 四、FKJ创作应用

**情绪 → 调性映射：**

| 情绪 | 推荐Camelot | 适用场景 |
|------|------------|----------|
| 孤独寂寞 | 8A, 9A | 深夜、电台 |
| 希望曙光 | 8B, 9B | 成长、励志 |
| 悲伤怀旧 | 7A, 11A | 回忆、告别 |
| 快乐庆祝 | 7B, 11B | 派对、高潮 |
| 平静治愈 | 5B, 6B | 放松、入睡 |

**FKJ风格推荐：**
- 韩风抒情：8A, 9A（小调为主）
- 治愈温暖：8B, 5B
- 成长励志：9B, 11B
