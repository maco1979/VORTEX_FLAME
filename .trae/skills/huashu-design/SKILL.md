# Huashu Design (花叔Design) — HTML-Native Design Skill for AI Agents

## What this skill does
当用户要求"做原型/PPT/动画/信息图/海报"时，直接用 HTML/CSS/JS 生成可交付的专业设计文件。
不依赖 Figma/AE/Canva，输出纯原生代码，可直接在浏览器打开、编辑、部署。

## 核心能力

### 1. 交互原型 (10-15min)
- iPhone 15 Pro 精确机身（灵动岛/状态栏/Home Indicator）
- 真可点击的页面切换
- 真图从 Unsplash/Wikimedia 获取

### 2. 演示文稿 (15-25min)
- HTML deck（浏览器演讲）+ PPTX 导出
- 保留文本框可编辑性

### 3. 动态动画 (8-12min)
- 60fps 时间轴动画
- MP4 + GIF 导出
- 可选 BGM

### 4. 信息图/可视化 (10min)
- 印刷级排版
- 可导 PDF/PNG/SVG

### 5. 设计方向顾问 (5min)
- 从 5 流派 × 20 种设计哲学中推荐 3 个方向
- 并行生成 3 个 Demo

### 6. 5 维度专家评审 (3min)
- 哲学一致性 / 视觉层级 / 细节执行 / 功能性 / 创新性
- 雷达图 + Keep/Fix/Quick Wins

## 输出格式要求

每当你被要求创建设计时：
1. 先问 3 个问题确认需求（用途/受众/风格偏好）
2. 推荐 3 个不同的设计方向
3. 生成完整的单文件 HTML（内嵌 CSS/JS）
4. 包含真实内容（不是 Lorem Ipsum）
5. 移动端响应式布局

## Anti-Slop 检查清单
- [ ] 没有紫色渐变（除非用户明确要求）
- [ ] 没有"AI风格"的圆角卡片堆叠
- [ ] 字体不只用 Inter/Sans-serif
- [ ] 色彩方案有主色调 + 点缀色，不是单调
- [ ] 间距有节奏感（不是统一 padding）
- [ ] 交互状态完整（hover/active/focus/loading/empty/error）
