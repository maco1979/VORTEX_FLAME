# 行业知识图谱技能

## 概述

基于UNESCO国际标准命名法和GB/T国家标准的完整行业知识图谱，为27个AI灵魂提供系统化行业知识分配框架。

## 核心知识体系

### 1. 基础学科体系 (UNESCO 1974)

七大基础学科：
- 数学 (Mathematics)
- 逻辑学 (Logic)
- 天文学和天体物理学 (Astronomy & Astrophysics)
- 地球科学和空间科学 (Earth & Space Sciences)
- 物理学 (Physics)
- 化学 (Chemistry)
- 生命科学 (Life Sciences)

### 2. 中国学科分类 (GB/T 13745-2009)

5大门类，62个一级学科，676个二级学科，2382个三级学科：
- A 自然科学 (110-190)
- B 农业科学 (210-240)
- C 医药科学 (310-360)
- D 工程与技术科学 (410-630)
- E 人文与社会科学 (710-910)

## 4层知识图谱架构

```
基础学科层 → 技术科学层 → 行业门类层 → 具体行业层
```

### 层1: 基础学科层
数学、物理学、化学、生物学、天文学、地球科学、逻辑学、系统科学、信息科学、认知科学

### 层2: 技术科学层
材料科学、计算机科学、电子科学、机械工程学、生物医学工程

### 层3: 行业门类层
20个经济门类 + 非经济门类

### 层4: 具体行业层
473个中类 + 1382个小类 + 事业单位 + 非营利组织

## 知识图谱关系类型

1. **依赖关系** - 行业对基础学科的直接依赖
2. **应用关系** - 基础学科在行业中的具体应用
3. **交叉关系** - 多学科交叉应用
4. **支撑关系** - 基础对行业的支撑
5. **发展关系** - 学科发展推动行业进步
