## 🌌 Project Overview

VORTEX_FLAME is an innovative 13-soul AI system that integrates specialized AI agents through an interactive galaxy interface. Users can find the most suitable AI expert for their needs through a complete path: requirements → industry → soul → tools → solution → application.

### 🎯 Key Features

- **🌌 Galaxy Visualization**: Interactive interface with black hole center and spiral arms based on astrophysics
- **🧠 13 AI Agents**: Specialized experts covering UNESCO 4-layer industry classification
- **💬 Real-time Chat**: Live conversations with deployed AI agents
- **🗺️ Intelligent Navigation**: Smart matching system from requirements to solutions
- **🚀 Production Ready**: Deployed agents with stable APIs and online demos

## 🔧 Installation & Setup

### Prerequisites
- Modern web browser
- Internet connection for AI agent APIs

### Quick Start
```bash
# Clone the repository
git clone https://github.com/maco1979/VORTEX_FLAME.git
cd VORTEX_FLAME

# Open the main interface
 open industry_knowledge_graph/index_user.html

# Or access the galaxy visualization
 open industry_knowledge_graph/hologram.html
```

### Development Setup
```bash
# Install dependencies (if any)
npm install  # or yarn install

# Start local server
python -m http.server 8000

# Access at http://localhost:8000
```
