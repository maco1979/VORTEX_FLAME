一、数学天花板数据集（S1阶段）

1. MathInstruct（核心，20万+高质量数学指令）
   https://huggingface.co/datasets/TIGER-Lab/MathInstruct
   - 点「Files and versions」→ 下载 jsonl 全量文件
   - 支持 git clone 批量拉取
   - 本地已有: soul_training_data/cezanne/cezanne_stage1_math_8k_v3.json (229911条)

2. MATH（数学推理金标准）
   https://huggingface.co/datasets/hendrycks/math

3. GSM8K / GSM100K（小学多步推理）
   GSM8K: https://huggingface.co/datasets/gsm8k
   GSM100K: https://huggingface.co/datasets/openai/gsm100k

4. AMC/AIME/奥林匹克题库（奥数天花板）
   https://huggingface.co/datasets/AI-MO/aimo-validation

5. ProofNet / Lean4 形式化证明（逻辑+数学终极）
   https://huggingface.co/datasets/ProofNet/proofnet


二、逻辑学天花板数据集（S2阶段，专治幻觉、三段论、因果）

1. LogiQA（中文逻辑天花板）
   https://huggingface.co/datasets/logiqa

2. FOLIO（一阶逻辑推理）
   https://huggingface.co/datasets/folio

3. ReClor（高阶逻辑）
   https://huggingface.co/datasets/reclor

4. ARC-Challenge（科学常识+逻辑）
   https://huggingface.co/datasets/ai2_arc


三、大模型底座语料（3000G核心来源，全免费）

1. BAAI 百川语料（国内天花板，中文为主）
   https://huggingface.co/datasets/BAAI/BAAI-Corpus

2. Common Crawl（全网高质量过滤语料，巨量）
   官方: https://commoncrawl.org/the-data/get-started/
   HF精选: https://huggingface.co/datasets/common_crawl

3. Wikipedia 全量（事实性天花板）
   官方: https://dumps.wikimedia.org/
   HF版: https://huggingface.co/datasets/wikipedia

4. RedPajama（复刻LLaMA训练语料，1.2万亿token）
   https://huggingface.co/datasets/togethercomputer/RedPajama-Data-1T

5. BookCorpus2 / PG19（书籍深度语料）
   BookCorpus2: https://huggingface.co/datasets/bookcorpus2
   PG19: https://huggingface.co/datasets/pg19


四、高质量SFT指令集（S2/S3阶段专用）

1. OpenOrca（GPT-4蒸馏天花板）
   https://huggingface.co/datasets/Open-Orca/OpenOrca

2. UltraChat（中文高质量对话）
   https://huggingface.co/datasets/thunlp/UltraChat

3. ShareGPT 高质量清洗版
   https://huggingface.co/datasets/anon8231489123/ShareGPT_V3_unfiltered_cleaned_split
