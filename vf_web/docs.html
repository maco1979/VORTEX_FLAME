文档 › 快速开始

快速开始

5 分钟启动

启动服务

cd VORTEX_FLAME
python vf_api_server.py

连接 Ollama

ollama pull hermes3:8b

打开界面

http://localhost:8765

API 参考

POST /api/ask
多专家提问，支持显式指定灵魂或自动路由
POST /api/orchestrate
跨知识库编排，支持 MoE / 自动 / 手动三种模式
GET /api/status
系统状态：Ollama连接、记忆条目、活跃灵魂
GET /api/souls
列出 14 专业知识库及能力
